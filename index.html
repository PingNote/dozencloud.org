歡迎來到伊達雲計畫

這是一個發展 ARM VPS 的計畫，現階段分兩部分 1. ODROID-XU4 2. Raspberry Pi B 主要以上面兩種設備建置

本專案未來會以開放原始碼授權釋出，現在正在研究合適的授權條款。

架構

使用 Openstack Kilo 以及 Docker LXC 建置 ARM 開發環境使用，提高每張 ARM 開發版使用率。使用 x86 電腦作為主控以及 SDN 控制單元，使用 ARM 開發版作為運算單元。利用 Docker 取代較為高負擔的虛擬化技術，提升使用者體驗。
